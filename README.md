# security-event-monitor

A Flask-based REST API for logging and monitoring security events.  
This project simulates a lightweight Security Operations Center (SOC) tool that tracks events like login attempts or network alerts.

#Features
- Create and retrieve security events through RESTful API endpoints  
- Filter events by severity or type  
- SQLite database for local data storage  
- JSON-based responses for easy integration  
- Extensible for JWT authentication or cloud deployment  

#Tech Stack
- **Python 3**
- **Flask** (API framework)
- **SQLite** (database)
- **REST API**
- **JSON**
